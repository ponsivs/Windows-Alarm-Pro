# Windows-Alarm-Pro
Windows Alarm Pro ensures you never miss important events by allowing you to set unlimited alarms with descriptions for waking up, taking medications, attending meetings, or taking breaks.
**Performance you can feel, Simplicity you will Appreciate**

A professional, feature-rich alarm system for Windows with voice announcements, recurring schedules, and intelligent notifications. Never miss an important event with this comprehensive time management solution.

## 📋 Features

### ⏰ Alarm Settings
- **Description** - Name your alarms
- **Date & Time** - Full date and time selection
- **Recurrence** - One Time, Daily, Weekly, Weekdays, Weekends, Monthly, Yearly, Custom Days
- **Categories** - General, Work, Personal, Meeting, Medication, Birthday, Reminder
- **Voice Announcements** - IGRF-Bhramma (Male) or IGRF-Saraswathi (Female) voices
- **Ring Duration** - 30 seconds to "Until Acknowledged"
- **Advance Warning** - 10-minute pre-notification
- **Popup Notifications** - Full-screen alerts when alarms trigger

### 🎯 Smart Features
- **Missed Alarm Detection** - Automatically reschedules recurring alarms if missed
- **Snooze Function** - 5-minute snooze with voice restart
- **Voice Test** - Preview voice announcements
- **System Tray Integration** - Minimize to tray with notifications
- **Keyboard Shortcuts** - Quick access to all features
- **Import/Export** - Save and restore alarm configurations
- **Automatic Logging** - Detailed activity logs for troubleshooting

### 🎨 Professional Interface
- Clean, modern GUI with IGRF branding
- Color-coded alarm status (upcoming, due, completed)
- Real-time date/time display
- Next alarm preview in status bar
- Category-based organization
- Visual recurrence indicators

## 🔧 Requirements

- **OS:** Windows 7/8/10/11 (32/64-bit)
- **Framework:** .NET Framework 4.5 or higher
- **PowerShell:** Version 5.1 or higher
- **Speech Recognition:** Windows Speech API (built-in)
- **Storage:** ~10 MB for logs and settings

## 📖 How to Use

### 1. Set an Alarm
- Enter a **Description** for your alarm
- Select **Date** and **Time**
- Choose **Recurrence** pattern
- Pick **Category** for organization
- Select **Voice** option (or No Voice)
- Set **Ring Duration**
- Enable/disable **Advance Notification** and **Popup Notification**

### 2. Manage Alarms
- **ADD ALARM** - Save current settings
- **CLEAR** - Clear description field
- **RESET** - Reset all fields to defaults
- **DELETE** - Remove selected alarm
- **ENABLE ALL** - Activate all alarms
- **IMPORT/EXPORT** - Backup/restore alarms
- **REFRESH** - Update the list
- **STOP** - Stop currently playing alarm

### 3. When Alarm Triggers
- **Popup appears** with alarm details
- **Voice announcement** (if selected)
- **ACKNOWLEDGE** - Stop and mark as complete
- **SNOOZE 5 MIN** - Postpone for 5 minutes

### 4. System Tray
- **Left-click** - Toggle window visibility
- **Right-click** - Show context menu
- **Double-click** - Restore window
- **Balloon tips** - Show minimize notifications

## ⚠️ Important Notes

- **Administrator rights** not required for normal operation
- **Speech synthesis** requires Windows Speech API (built-in)
- **Alarms persist** across application restarts
- **Missed alarms** are automatically rescheduled for recurring types
- **Sound volume** controlled separately from system volume
- **Logs** automatically rotate after 5MB
